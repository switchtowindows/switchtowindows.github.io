We want Windows to be your development environment.

We have been working hard to improve Windows 10 for developers. On this site, we are compiling dev tool comparisons, keyboard/trackpad shortcuts, how-to articles, software and service parity, hardware improvements, and as many compelling reasons as we can find for you to switch to Windows for your development environment.

Install Docker on Windows

Docker containers wrap a piece of software in a complete filesystem that contains everything needed to run: code, runtime, system tools, system libraries – anything that can be installed on a server. This guarantees that the software will always run the same, regardless of its environment. Learn how to install Docker for Windows.
